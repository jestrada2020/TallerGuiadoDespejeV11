Ejercicio 1 Despeje de variable R

Ecuación original: \(V=\frac{1}{3}\pi h(r^{2}+R^{2}+rR)\)

Variable a despejar: \(R\)

Concepto: Multiplicar ambos lados por 3 para eliminar la fracción.
\[ 3V = \pi h(r^{2}+R^{2}+rR) \]
Concepto: Dividir ambos lados por \(\pi h\).
\[ \dfrac{3V}{\pi h} = r^{2}+R^{2}+rR \]
Concepto: Reorganizar la ecuación para tener todos los términos con \(R\) en un lado y formar una ecuación cuadrática en términos de \(R\).
\[ R^2 + rR + r^2 - \dfrac{3V}{\pi h} = 0 \]
Concepto: Utilizar la fórmula general para resolver una ecuación cuadrática de la forma \(aR^2 + bR + c = 0\), donde \(a=1\), \(b=r\), y \(c = r^2 - \dfrac{3V}{\pi h}\).
\[ R = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] \[ R = \dfrac{-r \pm \sqrt{r^2 - 4(1)\left(r^2 - \dfrac{3V}{\pi h}\right)}}{2(1)} \]
Concepto: Simplificar el discriminante.
\[ R = \dfrac{-r \pm \sqrt{r^2 - 4r^2 + \dfrac{12V}{\pi h}}}{2} \] \[ R = \dfrac{-r \pm \sqrt{\dfrac{12V}{\pi h} - 3r^2}}{2} \]
Resultado:
\[ R = \dfrac{-r \pm \sqrt{\dfrac{12V - 3\pi h r^2}{\pi h}}}{2} \]
Ejercicio 2 Despeje de variable r

Ecuación original: \(C=2\pi r\)

Variable a despejar: \(r\)

Concepto: Dividir ambos lados por \(2\pi\).
\[ \dfrac{C}{2\pi} = \dfrac{2\pi r}{2\pi} \]
Concepto: Simplificar.
\[ \dfrac{C}{2\pi} = r \]
Resultado:
\[ r = \dfrac{C}{2\pi} \]
Ejercicio 3 Despeje de variable l

Ecuación original: \(P=2w+2l\)

Variable a despejar: \(l\)

Concepto: Restar \(2w\) de ambos lados.
\[ P - 2w = 2w + 2l - 2w \] \[ P - 2w = 2l \]
Concepto: Dividir ambos lados por 2.
\[ \dfrac{P - 2w}{2} = \dfrac{2l}{2} \] \[ \dfrac{P - 2w}{2} = l \]
Resultado:
\[ l = \dfrac{P - 2w}{2} \quad \text{o} \quad l = \dfrac{P}{2} - w \]
Ejercicio 4 Despeje de variable t

Ecuación original: \(I=Prt\)

Variable a despejar: \(t\)

Concepto: Dividir ambos lados por \(Pr\).
\[ \dfrac{I}{Pr} = \dfrac{Prt}{Pr} \]
Concepto: Simplificar.
\[ \dfrac{I}{Pr} = t \]
Resultado:
\[ t = \dfrac{I}{Pr} \]
Ejercicio 5 Despeje de variable h

Ecuación original: \(S=2\pi rh\)

Variable a despejar: \(h\)

Concepto: Dividir ambos lados por \(2\pi r\).
\[ \dfrac{S}{2\pi r} = \dfrac{2\pi rh}{2\pi r} \]
Concepto: Simplificar.
\[ \dfrac{S}{2\pi r} = h \]
Resultado:
\[ h = \dfrac{S}{2\pi r} \]
Ejercicio 6 Despeje de variable h

Ecuación original: \(A=\frac{1}{2}bh\)

Variable a despejar: \(h\)

Concepto: Multiplicar ambos lados por 2.
\[ 2A = 2 \times \frac{1}{2}bh \] \[ 2A = bh \]
Concepto: Dividir ambos lados por \(b\).
\[ \dfrac{2A}{b} = \dfrac{bh}{b} \] \[ \dfrac{2A}{b} = h \]
Resultado:
\[ h = \dfrac{2A}{b} \]
Ejercicio 7 Despeje de variable h

Ecuación original: \(V=\frac{1}{3}\pi r^{2}h\)

Variable a despejar: \(h\)

Concepto: Multiplicar ambos lados por 3.
\[ 3V = 3 \times \frac{1}{3}\pi r^{2}h \] \[ 3V = \pi r^{2}h \]
Concepto: Dividir ambos lados por \(\pi r^{2}\).
\[ \dfrac{3V}{\pi r^{2}} = \dfrac{\pi r^{2}h}{\pi r^{2}} \] \[ \dfrac{3V}{\pi r^{2}} = h \]
Resultado:
\[ h = \dfrac{3V}{\pi r^{2}} \]
Ejercicio 8 Despeje de variable m

Ecuación original: \(F=g\dfrac{mM}{d^{2}}\)

Variable a despejar: \(m\)

Concepto: Multiplicar ambos lados por \(d^2\).
\[ Fd^{2} = g\dfrac{mM}{d^{2}} d^{2} \] \[ Fd^{2} = gmM \]
Concepto: Dividir ambos lados por \(gM\).
\[ \dfrac{Fd^{2}}{gM} = \dfrac{gmM}{gM} \] \[ \dfrac{Fd^{2}}{gM} = m \]
Resultado:
\[ m = \dfrac{Fd^{2}}{gM} \]
Ejercicio 9 Despeje de variable b₁

Ecuación original: \(A=\frac{1}{2}(b_{1}+b_{2})\)

Variable a despejar: \(b_{1}\)

Concepto: Multiplicar ambos lados por 2.
\[ 2A = 2 \times \frac{1}{2}(b_{1}+b_{2}) \] \[ 2A = b_{1}+b_{2} \]
Concepto: Restar \(b_2\) de ambos lados.
\[ 2A - b_{2} = b_{1}+b_{2} - b_{2} \] \[ 2A - b_{2} = b_{1} \]
Resultado:
\[ b_{1} = 2A - b_{2} \]
Ejercicio 10 Despeje de variable r

Ecuación original: \(A=P+Prt\)

Variable a despejar: \(r\)

Concepto: Restar \(P\) de ambos lados.
\[ A - P = P + Prt - P \] \[ A - P = Prt \]
Concepto: Dividir ambos lados por \(Pt\).
\[ \dfrac{A - P}{Pt} = \dfrac{Prt}{Pt} \] \[ \dfrac{A - P}{Pt} = r \]
Resultado:
\[ r = \dfrac{A - P}{Pt} \]
Ejercicio 11 Despeje de variable v

Ecuación original: \(L^{2}=b\left({1 - \frac{v^{2}}{c^{2}}}\right)\)

Variable a despejar: \(v\)

Concepto: Dividir ambos lados por \(b\).
\[ \dfrac{L^{2}}{b} = 1 - \dfrac{v^{2}}{c^{2}} \]
Concepto: Restar 1 de ambos lados.
\[ \dfrac{L^{2}}{b} - 1 = - \dfrac{v^{2}}{c^{2}} \] \[ \dfrac{L^{2} - b}{b} = - \dfrac{v^{2}}{c^{2}} \]
Concepto: Multiplicar ambos lados por \(-1\).
\[ -\dfrac{L^{2} - b}{b} = \dfrac{v^{2}}{c^{2}} \] \[ \dfrac{b - L^{2}}{b} = \dfrac{v^{2}}{c^{2}} \]
Concepto: Multiplicar ambos lados por \(c^2\).
\[ c^{2}\left(\dfrac{b - L^{2}}{b}\right) = v^{2} \] \[ \dfrac{c^{2}(b - L^{2})}{b} = v^{2} \]
Concepto: Tomar la raíz cuadrada de ambos lados.
\[ \pm\sqrt{\dfrac{c^{2}(b - L^{2})}{b}} = v \]
Concepto: Simplificar la raíz cuadrada asumiendo \(c > 0\).
\[ \pm c\sqrt{\dfrac{b - L^{2}}{b}} = v \]
Resultado:
\[ v = \pm c\sqrt{\dfrac{b - L^{2}}{b}} \]
Ejercicio 12 Despeje de variable t

Ecuación original: \(s=\frac{1}{2}gt^{2}+v_{0}t\)

Variable a despejar: \(t\)

Concepto: Reorganizar la ecuación para formar una ecuación cuadrática en términos de \(t\).
\[ \frac{1}{2}gt^{2}+v_{0}t - s = 0 \]
Concepto: Utilizar la fórmula general para resolver una ecuación cuadrática de la forma \(at^2 + bt + c = 0\), donde \(a=\frac{1}{2}g\), \(b=v_0\), y \(c=-s\).
\[ t = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] \[ t = \dfrac{-v_{0} \pm \sqrt{(v_{0})^2 - 4(\frac{1}{2}g)(-s)}}{2(\frac{1}{2}g)} \]
Concepto: Simplificar el discriminante y el denominador.
\[ t = \dfrac{-v_{0} \pm \sqrt{v_{0}^2 + 2gs}}{g} \]
Resultado:
\[ t = \dfrac{-v_{0} \pm \sqrt{v_{0}^2 + 2gs}}{g} \]
Ejercicio 13 Despeje de variable q

Ecuación original: \(S=\dfrac{p}{q+p(1-q)}\)

Variable a despejar: \(q\)

Concepto: Multiplicar ambos lados por el denominador.
\[ S(q+p(1-q)) = p \]
Concepto: Distribuir \(p\) en el paréntesis.
\[ S(q+p-pq) = p \]
Concepto: Distribuir \(S\) en el paréntesis.
\[ Sq + Sp - Spq = p \]
Concepto: Mover los términos que no contienen \(q\) al otro lado.
\[ Sq - Spq = p - Sp \]
Concepto: Factorizar \(q\) del lado izquierdo.
\[ q(S - Sp) = p - Sp \]
Concepto: Dividir ambos lados por el factor de \(q\).
\[ q = \dfrac{p - Sp}{S - Sp} \]
Concepto: Opcional: Factorizar \(p\) en el numerador y \(S\) en el denominador.
\[ q = \dfrac{p(1 - S)}{S(1 - p)} \]
Resultado:
\[ q = \dfrac{p(1 - S)}{S(1 - p)} \]
Ejercicio 14 Despeje de variable r

Ecuación original: \(A=2\pi r(r+h)\)

Variable a despejar: \(r\)

Concepto: Distribuir \(2\pi r\).
\[ A = 2\pi r^2 + 2\pi rh \]
Concepto: Reorganizar la ecuación para formar una ecuación cuadrática en términos de \(r\).
\[ 2\pi r^2 + 2\pi hr - A = 0 \]
Concepto: Utilizar la fórmula general para resolver una ecuación cuadrática de la forma \(ar^2 + br + c = 0\), donde \(a=2\pi\), \(b=2\pi h\), y \(c=-A\).
\[ r = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] \[ r = \dfrac{-(2\pi h) \pm \sqrt{(2\pi h)^2 - 4(2\pi)(-A)}}{2(2\pi)} \]
Concepto: Simplificar el discriminante y el denominador.
\[ r = \dfrac{-2\pi h \pm \sqrt{4\pi^2 h^2 + 8\pi A}}{4\pi} \] \[ r = \dfrac{-2\pi h \pm \sqrt{4\pi(\pi h^2 + 2A)}}{4\pi} \] \[ r = \dfrac{-2\pi h \pm 2\sqrt{\pi(\pi h^2 + 2A)}}{4\pi} \]
Concepto: Simplificar la fracción dividiendo por 2.
\[ r = \dfrac{-\pi h \pm \sqrt{\pi(\pi h^2 + 2A)}}{2\pi} \]
Resultado:
\[ r = \dfrac{-\pi h \pm \sqrt{\pi^2 h^2 + 2\pi A}}{2\pi} \]

(Nota: Como \(r\) es típicamente un radio, debe ser no negativo. Elija la raíz positiva si es aplicable en el contexto.)

Ejercicio 15 Despeje de variable C

Ecuación original: \(P+N=\dfrac{C+2}{C}\)

Variable a despejar: \(C\)

Concepto: Multiplicar ambos lados por \(C\).
\[ C(P+N) = C+2 \] \[ CP + CN = C+2 \]
Concepto: Mover los términos con \(C\) a un lado y los términos sin \(C\) al otro.
\[ CP + CN - C = 2 \]
Concepto: Factorizar \(C\) del lado izquierdo.
\[ C(P + N - 1) = 2 \]
Concepto: Dividir ambos lados por el factor de \(C\).
\[ C = \dfrac{2}{P + N - 1} \]
Resultado:
\[ C = \dfrac{2}{P + N - 1} \]
Ejercicio 16 Despeje de variable D

Ecuación original: \(A=B\sqrt[3]{\dfrac{C}{D}}-E\)

Variable a despejar: \(D\)

Concepto: Sumar \(E\) a ambos lados.
\[ A+E = B\sqrt[3]{\dfrac{C}{D}} \]
Concepto: Dividir ambos lados por \(B\).
\[ \dfrac{A+E}{B} = \sqrt[3]{\dfrac{C}{D}} \]
Concepto: Elevar ambos lados al cubo para eliminar la raíz cúbica.
\[ \left(\dfrac{A+E}{B}\right)^3 = \dfrac{C}{D} \] \[ \dfrac{(A+E)^3}{B^3} = \dfrac{C}{D} \]
Concepto: Multiplicar ambos lados por \(D\).
\[ D \dfrac{(A+E)^3}{B^3} = C \]
Concepto: Multiplicar ambos lados por \(\dfrac{B^3}{(A+E)^3}\) para despejar \(D\).
\[ D = C \dfrac{B^3}{(A+E)^3} \]
Resultado:
\[ D = \dfrac{CB^3}{(A+E)^3} \]
Ejercicio 17 Despeje de variable R

Ecuación original: \(F=\dfrac{\pi PR^{4}}{8VL}\)

Variable a despejar: \(R\)

Concepto: Multiplicar ambos lados por \(8VL\).
\[ F(8VL) = \pi PR^{4} \] \[ 8FVL = \pi PR^{4} \]
Concepto: Dividir ambos lados por \(\pi P\).
\[ \dfrac{8FVL}{\pi P} = R^{4} \]
Concepto: Tomar la cuarta raíz de ambos lados.
\[ \sqrt[4]{\dfrac{8FVL}{\pi P}} = R \]
Resultado:
\[ R = \sqrt[4]{\dfrac{8FVL}{\pi P}} \]

(Nota: Como \(R\) es típicamente un radio, debe ser no negativo. Se ha elegido la raíz positiva.)

Ejercicio 18 Despeje de variable r

Ecuación original: \(V=V_{max}\left(1-\left(\dfrac{r}{r_0}\right)^2 \right)\)

Variable a despejar: \(r\)

Concepto: Dividir ambos lados por \(V_{max}\).
\[ \dfrac{V}{V_{max}} = 1-\left(\dfrac{r}{r_0}\right)^2 \]
Concepto: Restar 1 de ambos lados.
\[ \dfrac{V}{V_{max}} - 1 = -\left(\dfrac{r}{r_0}\right)^2 \] \[ \dfrac{V - V_{max}}{V_{max}} = -\left(\dfrac{r}{r_0}\right)^2 \]
Concepto: Multiplicar ambos lados por \(-1\).
\[ -\dfrac{V - V_{max}}{V_{max}} = \left(\dfrac{r}{r_0}\right)^2 \] \[ \dfrac{V_{max} - V}{V_{max}} = \left(\dfrac{r}{r_0}\right)^2 \]
Concepto: Tomar la raíz cuadrada de ambos lados.
\[ \sqrt{\dfrac{V_{max} - V}{V_{max}}} = \dfrac{r}{r_0} \]
Concepto: Multiplicar ambos lados por \(r_0\).
\[ r_0 \sqrt{\dfrac{V_{max} - V}{V_{max}}} = r \]
Resultado:
\[ r = r_0 \sqrt{1 - \dfrac{V}{V_{max}}} \]

(Nota: Como \(r\) es típicamente un radio, debe ser no negativo. Se ha elegido la raíz positiva.)

Ejercicio 19 Despeje de variable x

Ecuación original: \(W=b\left(\dfrac{a}{a+x}\right)^2\)

Variable a despejar: \(x\)

Concepto: Dividir ambos lados por \(b\).
\[ \dfrac{W}{b} = \left(\dfrac{a}{a+x}\right)^2 \]
Concepto: Tomar la raíz cuadrada de ambos lados.
\[ \sqrt{\dfrac{W}{b}} = \dfrac{a}{a+x} \]
Concepto: Tomar el recíproco de ambos lados.
\[ \dfrac{1}{\sqrt{\dfrac{W}{b}}} = \dfrac{a+x}{a} \] \[ \sqrt{\dfrac{b}{W}} = \dfrac{a+x}{a} \]
Concepto: Multiplicar ambos lados por \(a\).
\[ a\sqrt{\dfrac{b}{W}} = a+x \]
Concepto: Restar \(a\) de ambos lados.
\[ a\sqrt{\dfrac{b}{W}} - a = x \]
Concepto: Factorizar \(a\).
\[ x = a\left(\sqrt{\dfrac{b}{W}} - 1\right) \]
Resultado:
\[ x = a\left(\sqrt{\dfrac{b}{W}} - 1\right) \]
Ejercicio 20 Despeje de variable a

Ecuación original: \(W=b\left(\dfrac{a}{a+x}\right)^2\)

Variable a despejar: \(a\)

Concepto: Dividir ambos lados por \(b\).
\[ \dfrac{W}{b} = \left(\dfrac{a}{a+x}\right)^2 \]
Concepto: Tomar la raíz cuadrada de ambos lados.
\[ \sqrt{\dfrac{W}{b}} = \dfrac{a}{a+x} \]
Concepto: Multiplicar ambos lados por \(a+x\).
\[ (a+x)\sqrt{\dfrac{W}{b}} = a \] \[ a\sqrt{\dfrac{W}{b}} + x\sqrt{\dfrac{W}{b}} = a \]
Concepto: Mover los términos con \(a\) a un lado.
\[ a\sqrt{\dfrac{W}{b}} - a = -x\sqrt{\dfrac{W}{b}} \] \[ a\left(\sqrt{\dfrac{W}{b}} - 1\right) = -x\sqrt{\dfrac{W}{b}} \]
Concepto: Dividir ambos lados por el factor de \(a\).
\[ a = \dfrac{-x\sqrt{\dfrac{W}{b}}}{\sqrt{\dfrac{W}{b}} - 1} \]
Concepto: Simplificar la expresión.
\[ a = \dfrac{x\sqrt{\dfrac{W}{b}}}{1 - \sqrt{\dfrac{W}{b}}} \] \[ a = \dfrac{x\sqrt{W}}{\sqrt{b} - \sqrt{W}} \]
Resultado:
\[ a = \dfrac{x\sqrt{W}}{\sqrt{b} - \sqrt{W}} \]
Ejercicio 21 Despeje de variable v

Ecuación original: \(d=2\pi\left(\dfrac{vR}{0.52\cos(\theta)}\right)^{1/3}\)

Variable a despejar: \(v\)

Concepto: Dividir ambos lados por \(2\pi\).
\[ \dfrac{d}{2\pi} = \left(\dfrac{vR}{0.52\cos(\theta)}\right)^{1/3} \]
Concepto: Elevar ambos lados al cubo.
\[ \left(\dfrac{d}{2\pi}\right)^3 = \dfrac{vR}{0.52\cos(\theta)} \] \[ \dfrac{d^3}{8\pi^3} = \dfrac{vR}{0.52\cos(\theta)} \]
Concepto: Multiplicar ambos lados por \(0.52\cos(\theta)\).
\[ \dfrac{d^3 \times 0.52\cos(\theta)}{8\pi^3} = vR \] \[ \dfrac{0.52 d^3 \cos(\theta)}{8\pi^3} = vR \]
Concepto: Dividir ambos lados por \(R\).
\[ \dfrac{0.52 d^3 \cos(\theta)}{8\pi^3 R} = v \]
Resultado:
\[ v = \dfrac{0.52 d^3 \cos(\theta)}{8\pi^3 R} \]
Ejercicio 22 Despeje de variable θ

Ecuación original: \(d=2\pi\left(\dfrac{vR}{0.52\cos(\theta)}\right)^{1/3}\)

Variable a despejar: \(\theta\)

Concepto: Dividir ambos lados por \(2\pi\).
\[ \dfrac{d}{2\pi} = \left(\dfrac{vR}{0.52\cos(\theta)}\right)^{1/3} \]
Concepto: Elevar ambos lados al cubo.
\[ \left(\dfrac{d}{2\pi}\right)^3 = \dfrac{vR}{0.52\cos(\theta)} \] \[ \dfrac{d^3}{8\pi^3} = \dfrac{vR}{0.52\cos(\theta)} \]
Concepto: Multiplicar ambos lados por \(0.52\cos(\theta)\).
\[ \dfrac{0.52 d^3 \cos(\theta)}{8\pi^3} = vR \]
Concepto: Dividir ambos lados por \(\dfrac{0.52 d^3}{8\pi^3}\).
\[ \cos(\theta) = \dfrac{vR}{\dfrac{0.52 d^3}{8\pi^3}} \] \[ \cos(\theta) = \dfrac{8\pi^3 vR}{0.52 d^3} \]
Concepto: Aplicar la función arcocoseno (\(\cos^{-1}\)) a ambos lados.
\[ \theta = \cos^{-1}\left(\dfrac{8\pi^3 vR}{0.52 d^3}\right) \]
Resultado:
\[ \theta = \cos^{-1}\left(\dfrac{8\pi^3 vR}{0.52 d^3}\right) \]
